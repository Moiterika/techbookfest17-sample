## 注意事項

- vscodeの拡張機能利用を前提とした環境です
  - `.devcontainer/devcontainer.json`のcustomizations.vscode.extensionsを定義していません
  - 代わりに`my-todo/.vscode/extensions.json`に推奨の定義があります
